Instructions:

1. Run Init.bat on your primary storage computer with all of the archival storage drives connected
2. Copy this folder to a removable storage device
3. Plug the removable storage device into the devices you want to download items off of
4. Run Export.bat on all of the devices wanting to download from (leaving the external storage attached until the program completes)
5. Copy the contents of this folder back onto your primary storage computer
6. All of the media files across all of your devices should now be on your primary storage computer
