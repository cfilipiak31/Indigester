Instructions:

1. Run Init.bat on your primary storage computer with all of the archival storage drives connected (that you plan on continuing to use)
2. Copy this folder to a removable storage device
3. Plug the removable storage device into the devices you want to download items off of
4. Run Import.bat on all of the devices wanting to download from (leaving the external storage attached until the program completes)
5. Plug the drive back into your primary storage computer, and run Import.bat on the copy of the folder currently existing on one of the computers drives
6. All of the media files across all of your devices should now be on your primary storage computer
